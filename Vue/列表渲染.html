列表渲染 ​

对于数组、对象这样的结构化数据，若要将它们的元素提取出来放入HTML中，势必会做很多重复劳动，列表渲染就是为了简化这个过程而生。

渲染数组 ​

下面是列表渲染数组的示例。

vue
<script setup>
const arr =[1,2,3]
</script>
<template>
<p v-for="x in arr">{{x}}</p>
</template>
<script setup>
const arr =[1,2,3]
</script>
<template>
<p v-for="x in arr">{{x}}</p>
</template>

渲染对象 ​

下面是列表渲染对象的示例。

vue
<script setup>
const obj ={a:1,b:2}
</script>
<template>
<p v-for="(v, k) in obj">{{k}}:{{v}}</p>
</template>
<script setup>
const obj ={a:1,b:2}
</script>
<template>
<p v-for="(v, k) in obj">{{k}}:{{v}}</p>
</template>

注意，括号中的顺序是值的形参、键的形参，不要用反了。
