# 文本插值

文本插值比较简单，就是使用双大括号将变量包裹住，vue会自动解析出该变量的值渲染到页面，如下示例：

```vue
<script setup>
const a = 1
</script>

<template>
  {{ a }}
</template>
```

文本插值更强大的功能是可以使用任何JavaScript表达式：

```vue
<script setup>
const a = 1
const str = "hello,world"
const arr = [1,2,3]
</script>

<template>
  {{ a }}
{{ a + 1 }}
{{a%2==0?'偶数' : '奇数' }}
{{str.split(",")}}
{{arr.map(x=>x+1)}}
</template>
```
