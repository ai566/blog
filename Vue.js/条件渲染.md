# 条件渲染

条件渲染的意思是根据一定的判断条件确定是否加载这个元素，会用到三个指令：`v-if`、`v-else-if`、`v-else`。

## 单分支条件渲染

```vue
<script setup>
const score = 80
</script>
<template>
<p v-if="score>=80">优秀</p>
</template>
```

## 双分支提交渲染

```vue
<script setup>
const score = 60
</script>
<template>
<p v-if="score>=80">优秀</p>
<p v-else>不优秀</p>
</template>
```

## 多分支条件渲染

```vue
<script setup>
const score = 50
</script>
<template>
<p v-if="score>=80">优秀</p>
<p v-else-if="score>=60">及格</p>
<p v-else>不及格</p>
</template>
```
